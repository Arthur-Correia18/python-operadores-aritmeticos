# 🧮 Operadores Aritméticos em Python

## 📘 Sobre o Projeto
Projeto desenvolvido durante meus estudos em Python com foco em operadores aritméticos e operadores de atribuição.

## 🧠 Conceitos aplicados

- Soma (+)
- Subtração (-)
- Multiplicação (*)
- Divisão (/)
- Divisão inteira (//)
- Resto da divisão (%)
- Operadores de atribuição (+=, -=, *=, /=)
- Cálculo de média

## 💻 Funcionalidade

O projeto contém exemplos práticos de:

- Operações matemáticas básicas
- Cálculo de média entre três números
- Incremento e decremento de variáveis
- Cálculo de divisão inteira e resto

## ▶️ Exemplo de execução

Digite o primeiro número: 10  
Digite o segundo número: 5  

Resultados:  
soma: 15  
subtracao: 5  
multiplicacao: 50  
divisao: 2.0
